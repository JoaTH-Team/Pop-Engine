class ChannelPair

package flixel.addons.display.waveform

implements IFlxDestroyable

import flixel.addons.display.waveform.FlxWaveformBuffer

@:directlyUsed@:allow(flixel.addons.display.waveform.FlxWaveformBuffer)

A ChannelPair is a helper class containing audio data for both audio channels.

Constructor

new(left:Float32Array, right:Float32Array)

Creates a new ChannelPair from two Float32Array instances.

Parameters:

left

Float32Array instance containing audio data for the left channel

right

Float32Array instance containing audio data for the right channel

Methods

destroy():Void

Nulls all data related to the ChannelPair

getChannelData(channel:Int):Null<Float32Array>

Returns a Float32Array containing raw audio data for the specified audio channel.

Parameters:

channel

The audio channel to get audio data for

Returns:

A Float32Array or null if there's no audio data for the specified channel.
