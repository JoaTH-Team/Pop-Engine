class FlxCache

package flixel.system

@:directlyUsed@:access(flixel.system.frontEnds.BitmapFrontEnd)

A custom Asset Cache system designed to replace OpenFL's/Lime's Asset Cache system.

Constructor

@:value({ enabled : true })new(enabled:Bool = true)

Creates a new cache instance. You can optionally disable caching by passing false to the constructor.

Variables

enabled:Bool

If this is set to false, none of the cache logic will run. It's a global toggle for whether or not anything should be cached.

Methods

clear():Void

Completely clears everything currently stored in the cache. Useful if you want to fully unload memory or reset.

list():Array<String>

Returns a list of all asset keys stored in the cache. Keys from all three asset types (bitmap, sound, font) are included.
