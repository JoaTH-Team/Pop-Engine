Pop Engine - Home Page

Pop Engine is a scripted game engine.

Features:

  • Lua/HScript scripting game
  • Able to create more callback if the current base one is not enough (by using HScript only)
  • Simple Manager State to load game content
  • A simple crash handler (it will show a pop-up and then create a log file from it)
